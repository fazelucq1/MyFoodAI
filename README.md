# MyFoodAI - Generatore di Ricette

## Setup
1. Clona il repo
2. Imposta `GEMINI_API_KEY` come variabile d'ambiente
3. `pip install -r server/requirements.txt`
4. `python server/server.py`
5. Accedi a `http://localhost:5000`
